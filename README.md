🎵 Spotify Clone

A pixel-perfect clone of the **Spotify Web Player** UI, built using only **HTML** and **CSS** — no frameworks, no JavaScript.

---

✨ Features

- 🎨 Spotify-inspired dark theme UI
- 📚 Sidebar with navigation and Your Library section
- 🃏 Music cards with album art, title, and description
- ▶️ Bottom music player bar with a styled progress bar
- 📌 Sticky top navigation bar
- 📱 Responsive design — sidebar hides on smaller screens
- 🔤 Montserrat font & Font Awesome icons

---

🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Page structure & layout |
| CSS3 | Styling, Flexbox, media queries |
| Font Awesome 6 | Icons (home, search, play controls) |
| Google Fonts | Montserrat font family |

---

📁 Project Structure

```
spotify-clone/
├── index.html        # Main HTML file
├── style.css         # All styles
├── Images/           # Album art, icons, logos
└── README.md
```

---

🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/AnmolBhardwaj0-0/spotify-clone.git
   ```

2. Open the project folder:
   ```bash
   cd spotify-clone
   ```

3. Open `index.html` in your browser — that's it, no installs needed!
